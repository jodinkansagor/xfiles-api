Character API

**Documentation Currently Under Construction**


ENDPOINTS


/api/v1/characters

Gets a list of characters in alphabetical order. Without any query options, returns all characters in one array. Query options include page, perPage, and category


Examples:

https://xfiles-api.herokuapp.com/api/v1/characters?category=Main_characters


https://xfiles-api.herokuapp.com/api/v1/characters?perPage=20&page=3


    Categories include:

  • Monster of the Week
  • Main characters
  • Syndicate
  • Informants
  • FBI personnel
  • Deceased people


    • /api/v1/characters/:name

    Returns a character by name


    Example:

    https://xfiles-api.herokuapp.com/api/v1/characters/Flukeman
