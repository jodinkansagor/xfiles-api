Endpoints

/api/v1/characters

Gets a list of characters. Without any query options, returns all characters in one array. Query options include page, perPage, and category

    Categories include:
  • Monster of the Week
  • Main characters
  • Syndicate
  • Informants
  • FBI personnel
  • Deceased people

    • /api/v1/characters/:name

    Returns a character by name
